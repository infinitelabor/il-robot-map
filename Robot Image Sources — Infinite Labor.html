How to use this guide

  1. Create an images/ folder inside your GitHub repo, then subfolders per sub-type (e.g. images/delta-scara/)
  2. For each robot, visit the source URL, download the image, and save it using the exact filename shown
  3. Images should ideally be 800–1200px wide, landscape or square. The UI crops them to 220px height at full width.
  4. To swap a photo later: just replace the file. No code changes needed. Same filename, same folder.
  5. For Wikimedia Commons URLs: click the image on the page → click "Download" → choose "Original file" or a medium resolution
